  • Twitter Bot Detection

    • • The primary focus of this project was to explore non-traditional ways of detecting twitter bots using Machine Learning. The fact that twitter allows automation (bots), made this problem challenging. Our work was centered on detecting spam bots along with a probabilistic confidence
    • • In addition to bag of words, TF-IDF and doc2Vec features based on user tweets and aggregated user tweet content; we also used several derived features like number of hashtags, tweet length, URLs per tweets, etc. Dimensionality reduction and feature selection was achieved using Truncated Singular Value Decomposition
    • • Logistic Regression, Support Vector Machines, Random Forest, Multilayer Perceptron Neural Networks and Stochastic gradient descent were used for this project
    • • An incremental knowledge version (online) stochastic gradient descent was used to analyze the impact of providing aggregated tweet content by time snapshots to the model, in order to facilitate early detection with confidence details.     • • Eventually an all-encompassing model that used derived features from user account metadata, aggregated tweet content for user and available account metadata was build using Support Vector Machines. This model achieved an accuracy of 100% for the botometer datasets
